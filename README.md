## Variants

1. **Sercali Dark (High Blur)**
   Transparent background for stronger blur effects while floating surfaces stay readable.
2. **Sercali Dark (Soft Blur)**
   Subtle 90% opacity background for depth without heavy transparency.
3. **Sercali Dark (Opaque)**
   Solid Deep Basalt for maximum focus and contrast.
4. **Sercali Light (Opaque)**
   Warm paper-like light mode with an ancient-tech tone.

## Installation

### Install from Zed Extensions

1. Open Zed.
2. Run `cmd+shift+p`.
3. Choose `zed: extensions`.
4. Search for `Sercali` and install.
5. Select a Sercali variant from the theme picker.
